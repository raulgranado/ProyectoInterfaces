Ventana principal

En la ventana principal se encuentra la gestión de la base de datos (Añadir, Buscar, Ver todo)

La opción de Añadir abrirá una nueva ventana en la que se nos pedirá una serie de datos

La opción de Buscar permite encontrar en la base de datos los libros que reúnan los requisitos

La opción de Ver todo listará todos los libros en la base de datos
